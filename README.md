# 💊 Medicine Management System

## 📌 Project Description

Medicine Management System is a web-based application developed to manage medical store operations efficiently. It helps in managing medicines, orders, users, and payments in a digital way.

---

## 🚀 Features

* 👤 User Registration & Login
* 🛒 Add to Cart & Wishlist
* 💳 Online Payment System
* 📦 Order Placement & Tracking
* 🧾 Bill Generation
* 🔐 Admin Panel
* 📊 Manage Products, Users & Orders
* 📩 Email Notification System

---

## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Server: WAMP Server

---

## 📂 Project Structure

* `/Medico` → User side website
* `/admin_area` → Admin panel
* `/includes` → Database & common files
* `/images` → Images
* `/uploads` → Uploaded files

---

## ⚙️ Installation Steps

1. Install WAMP/XAMPP Server
2. Copy project into `www` folder
3. Import database (`mystore.sql`) into MySQL
4. Start Apache & MySQL
5. Open browser and run:
   http://localhost/MedicoProject
